# Uber Reimagined – Taxi/Cab Booking Mobile App

## 📌 Project Overview
Uber Reimagined is a modern Taxi and Cab Booking Mobile App UI/UX design created in Figma. The app is designed to provide users with a seamless ride-booking experience, allowing them to easily search for destinations, book rides, track drivers, manage trips, and make secure payments through an intuitive and visually appealing interface.

## 🎯 Design Goal
The goal of this project was to redesign and enhance the ride-booking experience by focusing on usability, accessibility, and modern design principles. The interface aims to simplify the booking process while providing users with real-time ride information and smooth navigation.

## 📱 Pages Designed
- Splash Screen
- Onboarding Screens
- Login & Registration Screens
- Home Screen
- Ride Search Screen
- Location Selection Screen
- Driver Selection Screen
- Ride Confirmation Screen
- Live Tracking Screen
- Payment Screen
- Trip History Screen
- User Profile Screen
- Interactive Prototype Flow

## ✨ Key UX Decisions
- Quick and hassle-free ride booking flow
- User-friendly navigation and screen transitions
- Clear ride details and fare information
- Real-time ride tracking experience
- Easy payment and trip management
- Mobile-first design approach
- Consistent user journey across all screens

## 🎨 Color & Typography
- Modern and professional color palette
- High-contrast UI elements for readability
- Clean and accessible typography
- Consistent visual hierarchy and spacing

## 🛠 Tools Used
- Figma
- Auto Layout
- Components
- Interactive Prototyping
- Icons & Design Assets

## 🚀 Outcome
The final design delivers a smooth and efficient cab booking experience, helping users quickly book rides, track their journeys, manage payments, and access trip information through an engaging and modern interface.
